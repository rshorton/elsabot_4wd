# Elsabot 4WD Robot

Top-level bring up scripts for the 4WD base version of the Elsabot robot.  

Some parts of these scripts and configuration files are from the Linorobot2 project:  https://github.com/linorobot/linorobot2

The firmware for the base microcontroller is based on the linorobot2_hardware project.  See this fork for the firmware used for the Elsabot 4WD base: https://github.com/rshorton/linorobot2_hardware.
